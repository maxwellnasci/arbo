# 🎯 Estratégia de Consultoria — Maxwell

## Posicionamento
Não sou um consultor genérico de IA.
Sou o cara que constrói software para assessorias esportivas no Brasil.
O Arbo é a prova. O processo é o diferencial.

## O Negócio

### Curto prazo — Escalar o Arbo
- Modelo: app personalizado por assessoria
- Ticket: R$3-5k implantação + R$500-1.500/mês manutenção
- Clientes alvo: professores de corrida, CrossFit, Hyrox, funcional
- Vantagem: já existe, já funciona, cliente real usando

### Parceria com professor de CrossFit/Hyrox
Divisão clara de responsabilidades:
- Maxwell: tecnologia, desenvolvimento, manutenção do app
- Professor: lógica dos treinos, CrossFit, Hyrox, corrida personalizada
- Modelo: sociedade ou revenue share por cliente fechado
- Por que funciona: cada um faz o que sabe, ninguém tenta fazer tudo

### Médio prazo — Consultoria de ticket alto
- Modelo: construir o app do cliente do zero usando o PLAYBOOK
- Ticket: R$15-30k por projeto
- Prazo: 30 dias (comprovado no Arbo)
- Diferencial: processo documentado, time de IA, entrega previsível

## O Case — Arbo em números
- Prazo: ~30 dias do zero ao ar
- Stack: React + TypeScript + Supabase + Vercel
- Qualidade: Lighthouse 96 Performance, 100 SEO, 100 Best Practices
- Nota geral: 8.75/10
- Features: auth, turmas, treinos, check-in, PRs, chat, PWA, CI/CD
- Time: Claude + Gemini + DeepSeek + Maxwell no comando

## Próximos 90 dias

### Mês 1 — Validar
- [ ] Professor testa o Arbo em produção com alunos reais
- [ ] Coletar feedback e métricas (horas economizadas, retenção)
- [ ] Fechar parceria com professor de CrossFit/Hyrox
- [ ] Adaptar Arbo para CrossFit (primeiro cliente da parceria)

### Mês 2 — Escalar
- [ ] 2-3 clientes pagantes (assessorias esportivas)
- [ ] Documentar case com números reais
- [ ] Criar página de apresentação simples
- [ ] LinkedIn atualizado com o case

### Mês 3 — Crescer
- [ ] 5+ clientes ativos
- [ ] Primeiro projeto de consultoria de ticket alto fechado
- [ ] Avaliar se busca emprego híbrido ou continua consultoria

## O que NÃO fazer
- Não tentar ser plataforma SaaS agora — complexidade alta, receita baixa
- Não vender "automação genérica com IA" — commodity
- Não aceitar 15 clientes a R$397 — é emprego ruim, não negócio
- Não construir mais features sem feedback do cliente real

## Lição do Fable 5
"Escolha entre produto ou portfólio — o meio-termo mata.
Consultor genérico de IA compete com o mundo inteiro.
O cara do software de assessoria esportiva compete com ninguém."
